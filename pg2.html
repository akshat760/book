Preface

Computer Applications for Class X covers the complete syllabus of Class X issued by CBSE. The higher education community is planning for a world in which information technology (IT) will be the base and the very institution of higher education is sure to change. Of course, IT can probably be used to improve higher education, as it is exceedingly flexible and we faced numurous choices about how best to apply it.
This book deals with the Internet, Internet Services, HTML programming and Cyber Ethics.

The current syllabus also introduces two programming languages called Scratch and Python. Out of the two languages you have to opt any one language. The programminglanguage Scratch is a multimedia based language that supports games, animations, story-based animations, etc, for all age groups. But python is a general purpose high-level programming language provided as an option to children with special needs.
This book has been written in simple, lucid clear and user-friendly style, such that every students may find the subjects matter interesting, easy and informative.


A CD(.../source_X) which includes chapterwise application files of HTML programming, the script mode for scratch and source code of python can be made available on demand. for easy practice, explore the directories/ folders and test them in your own computer.

Many solved and unsolved examples including MCQs(Multiple Choice Questions) with answers, Review questions and lab exercises provided at the end of each chapter which will enablw students to test their ability.
